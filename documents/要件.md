実際にCloudflare上で行う操作はこれです。

1. Cloudflare Dashboard にログイン
2. Workers & Pages を開く
3. Create application を押す
4. Pages を選ぶ
5. Connect to Git を選ぶ
6. GitHub を認可する
7. 公開したいリポジトリを選ぶ
8. Project name を決める
9. Production branch を main にする
10. Build settings を設定する
11. Save and Deploy を押す
12. 発行された https://<project-name>.pages.dev/ を確認する
